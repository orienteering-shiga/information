# 淡海ロゲイニング2024 「走る君へ」

およそ1000年前の貴族が栄華を誇った平安時代、和歌や新しい文学が多く生まれました。大河ドラマ「光る君へ」（NHK）ではこの時代、貴族の生き方を描いた源氏物語を生み出した紫式部の生涯が描かれました。そんな源氏物語が生まれたゆかりの地、琵琶湖のほとりの大津市でロゲイニングを開催します。平安時代の足跡を追いかけてみませんか。
